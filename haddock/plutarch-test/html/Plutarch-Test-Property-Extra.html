l is a left inverse of r

See https://en.wikipedia.org/wiki/Inverse_function#Left_inverses

Like prop_haskEquiv, you want to call this with TypeApplications specifying the value of e. For example,

>>> prop_leftInverse
  @'OnPEq
  mapJoin
  mapSplit
  $ mapOf (pairOf integer integer) rational
